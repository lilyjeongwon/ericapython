
# 🌱 Erica's 작물 키우기 게임

5가지 작물을 키우는 파이썬 pygame 게임입니다.

## 작물 종류
- 해바라기
- 상추
- 완두콩
- 감자
- 용과
  
## 실행 방법
```bash
pip install pygame
python src/main.py
```

## 업데이트
- esc: 게임 종료
- F3: 돈추가
- venv -> main.py 로 실행
