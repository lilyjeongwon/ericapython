🌾 농장 시뮬레이터 게임

파이썬과 Pygame으로 만든 재미있는 농장 시뮬레이터 게임입니다!

다양한 작물을 재배하고, 도구를 업그레이드하며, 농장을 발전시켜보세요!

📱

반응형 디자인

휴대폰, 태블릿, PC 모든 기기에서 완벽하게 작동

🌱

다양한 작물

해바라기, 토마토, 옥수수 등 다양한 작물 재배

🛠️

도구 시스템

물뿌리개와 낫을 업그레이드하여 효율성 증대

🎮 게임 다운로드

📦 ZIP 파일 다운로드

⚙️ 설치 및 실행 방법

  1. Python 3.8 이상 설치 (python.org에서 다운로드)
  2. Pygame 라이브러리 설치: pip install pygame
  3. 다운로드한 파일 압축 해제
  4. 터미널에서 python scripts/main.py 실행

🎯 게임 조작법

  • 마우스 클릭: 작물 심기, 물주기, 수확하기
  • 상점 버튼: 씨앗과 도구 구매
  • 설정 아이콘: 볼륨 조절
  • F1 키: 빠른 성장 모드 토글

🌟 개발자 정보
